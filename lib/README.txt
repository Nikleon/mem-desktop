To successfully build the project, you must download the following dependencies into this lib folder:

commons-validator-1.6.jar
(from: http://mirrors.gigenet.com/apache//commons/validator/binaries/commons-validator-1.6-bin.zip)

commons-lang3-3.7.jar
(from: http://mirrors.koehn.com/apache//commons/lang/binaries/commons-lang3-3.7-bin.zip)
